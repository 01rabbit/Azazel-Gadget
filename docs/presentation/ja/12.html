Core Capabilities

FUNCTIONAL MODULES

SYSTEM_CAPABILITIES_V2

Traffic Monitoring

ネットワークフローをリアルタイムに可視化し、接続端末・プロトコル・通信量異常を即時に把握します。

MOD_01

Anomaly Detection

SuricataベースのIDSエンジンで、MITM・ポートスキャン・既知の攻撃パターンをシグネチャで検知します。

MOD_02

Dynamic Control

`tc` と `iptables` による能動的なトラフィック制御で、悪性IPからの通信を自動で遅延または遮断します。

MOD_03

Decoy Generation

OpenCanary（SSH/HTTP/）を起動して攻撃者を誘導し、行動パターンを記録します。

MOD_04

Risk Assessment

Tactics Engineが観測行動に基づいて脅威度を評価し、接続端末ごとにリスクスコアを付与します。

MOD_05

Internal Telemetry

内部JSONエンジンで自己完結型のログ集約を実施し、通知はローカルハードウェアのみに送信します。外部依存はありません。
