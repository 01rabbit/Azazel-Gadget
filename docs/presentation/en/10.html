OPERATIONAL MODES

DEFINING BEHAVIOR & EXPOSURE SURFACES

SHIELD

DEFAULT

BLOCK INBOUND on wlan0. Allows usb0 outbound. Safe for daily carry. No decoy exposure.

PORTAL

Acts as TRANSPARENT GATEWAY. NAT enabled for usb0 clients. Zero friction onboarding. No decoy exposure.

SCAPEGOAT

DECOY ACTIVATED. Exposes OpenCanary ports. Isolated in `az_canary` netns. Trap & Trace.

CONTROL LOGIC & INVARIANTS

SINGLE SOURCE OF TRUTH

/etc/azazel/mode.json

SECURITY INVARIANT

NEVER allow wlan0 -> usb0 inbound.

CLI: sudo azctl mode set {portal|shield|scapegoat}
